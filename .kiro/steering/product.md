# Product Overview

AIGens is a multi-provider AI platform that aggregates various AI models and services into a unified API. The platform allows users to interact with different AI providers (Anthropic, OpenAI, Together.ai, OpenRouter, DeepSeek, etc.) through a single interface.

## Core Features

- **Multi-Provider AI Access**: Unified access to models from Anthropic, OpenAI, Together.ai, OpenRouter, DeepSeek, and others
- **Cost Management**: Real-time cost calculation, credit system, and usage tracking
- **Chat Interface**: Conversational AI with support for text, images, and file attachments
- **User Management**: OAuth authentication (Google, Microsoft, GitHub), user profiles, and access control
- **Payment Integration**: Stripe and BTCPay integration for credit purchases
- **Model Analytics**: Usage statistics, performance metrics, and cost analysis

## Business Model

- Credit-based system where users purchase credits to use AI models
- Markup pricing on top of provider costs
- Support for both direct provider integration and aggregator services
- Subscription-based access to different model tiers

## Target Users

- Developers and businesses needing AI model access
- Users wanting to compare different AI providers
- Organizations requiring cost-effective AI solutions with usage tracking
