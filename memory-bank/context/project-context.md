# Project Context - Aigens Backend

## Overview
Aigens Backend is a Node.js application that provides backend services for the AIGens platform. The project uses Express.js as the web framework and Sequelize as the ORM for MySQL database interaction.

## Main Technologies
- Node.js (>=16.0.0)
- Express.js
- Sequelize (ORM)
- MySQL
- Passport.js (Authentication)
- Swagger (API Documentation)

## Key Dependencies
- @anthropic-ai/sdk: Anthropic AI integration
- openai: OpenAI integration
- passport-*: Authentication strategies (Google, Microsoft, GitHub)
- swagger-jsdoc/swagger-ui-express: API documentation

## Development Environment
- Nodemon for hot-reloading
- Jest for testing
- ESLint for linting
- Supertest for API testing
